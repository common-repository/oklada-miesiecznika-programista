== Description ==
Widget dodający okładkę magazynu Programista. Zawsze linkuje do najnowszego wydania magazynu.

== Installation ==
1. Plik programista-widget.php należy wrzucić do katalogu wp-content/plugins lub w panelu admina użyć funkcji Plugins (Wtyczki) -> Add new (Dodaj nową) -> Upload Plugin (Wyślij wtyczkę na serwer).
2. Po wgraniu wtyczki należy ją aktywować w zakładce Plugins (Wtyczki).
3. Na koniec należy skonfigurować wyświetlanie widgetu w zakładce Appearance (Wygląd) -> Widgets (Widgety). Po poprawnej aktywacji pluginu powinien pojawić się bloczek "Programista - okładka".
